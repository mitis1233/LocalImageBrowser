# 🌌 終極本地圖片瀏覽器 (Local Image Browser)

這是一個**完全免安裝、零伺服器依賴**的單檔 (Single-file) 本地圖片瀏覽器。只需下載一個 HTML 檔案並在瀏覽器中點擊兩下，即可瞬間載入擁有數千甚至數萬張圖片的超大容量資料夾。專為漫畫閱讀、大量圖庫整理與極致沉浸看圖所打造。

## ✨ 核心特色功能

* 🚀 **極限效能與防當機**：導入 `IntersectionObserver` 無限捲動與延遲載入 (Lazy Loading) 技術。即使選取十萬張圖片的資料夾，也能在 1 秒內掃描完成，並只渲染畫面可見的圖片，徹底拒絕記憶體溢出 (OOM)。
* 📁 **自動穿透深層資料夾**：支援 HTML5 `webkitdirectory`，選取母資料夾（如 `Downloads`）後，程式會自動遍歷並抓出所有深層子資料夾內的圖片。
* 🔍 **神級看圖手感 (Zoom-to-Pointer)**：
  * **指針追蹤縮放**：重寫滑鼠滾輪邏輯，**滾輪會精準朝著你滑鼠游標所在的位置進行放大/縮小**，並採用 1.15 倍乘法平滑縮放，體驗媲美專業看圖軟體。
  * **自由拖曳平移**：圖片放大後，按住滑鼠左鍵即可無縫拖曳 (Pan) 查看細節，已完美修復連點反藍 BUG。
* 📐 **主介面與燈箱「狀態分離」**：
  * **三種滿版模式**：提供「自動滿版」、「高度滿版」、「寬度滿版 (漫畫模式)」。
  * **無縫接軌**：主介面與全螢幕燈箱具備獨立的縮放記憶。在主畫面調整的網格大小與縮放倍率，進入燈箱時會自動保護切換為最適合的滿版模式；退出燈箱時，外層排版完美還原，絕不跑位。
  * **自訂縮放突破限制**：提供 1~150% 快速滑桿，並支援手動輸入突破限制（如 500%、1000%）的精準縮放。
* 👁️ **極致沉浸與無干擾 UI**：
  * **一鍵隱藏介面**：專屬的「👁️ 顯示/隱藏介面」開關，一鍵清除所有檔名文字與懸浮按鈕，把螢幕 100% 還給圖片。
  * **自動隱藏頂部列**：向下捲動網頁時頂部選單自動隱藏，向上滑動即時喚出。
  * **隱形盲點切換**：燈箱左右兩側 25% 區域為 99% 透明的巨大切換按鍵；右上角設有隱形感應大範圍「關閉 (X)」按鈕，滑鼠甩動過去即浮現。
* 🔗 **一鍵開啟原始大圖**：燈箱左上角的圖片標題為動態超連結，點擊即可在瀏覽器「新分頁」中以 100% 最純粹的原始解析度檢視或另存圖片。

## 🛠️ 使用方式

1. 下載本專案中的 `Local Image Browser.html` 檔案。
2. 對著檔案點擊兩下，使用 Google Chrome、Microsoft Edge、Brave 等現代瀏覽器開啟。
3. 點擊畫面左上角的 **「📁 選擇資料夾」**。
4. 在彈出的系統視窗中，選擇你存放圖片的母資料夾（例如 `D:\Downloads`）。
   > *⚠️ 瀏覽器安全性提醒：此時瀏覽器會跳出「要上傳檔案到這個網站嗎？」的警告視窗。請放心點選「上傳」，本程式**完全純前端運行於本地端**，絕對不會將圖片上傳至網路，也不會在硬碟中製造任何暫存拷貝檔案，僅是在記憶體中建立唯讀通道。*
5. 開始享受極致流暢的看圖體驗！

## ⌨️ 快捷鍵與滑鼠操作指南

### 全螢幕燈箱 (Lightbox) 操作
| 操作方式 | 功能說明 |
| :--- | :--- |
| **滑鼠滾輪** | **指針追蹤縮放** (朝著游標位置放大/縮小當前圖片) |
| **按住滑鼠左鍵** | 拖曳平移已放大的圖片 (觀看漫畫或長條圖神器) |
| **點擊畫面左側 25% 區域** | 切換至上一張圖片 |
| **點擊畫面右側 25% 區域** | 切換至下一張圖片 |
| **點擊左上角標題** | 於新分頁開啟 100% 原始圖片 |
| **鍵盤 `→` (右方向鍵)** | 切換至下一張圖片 |
| **鍵盤 `←` (左方向鍵)** | 切換至上一張圖片 |
| **鍵盤 `ESC`** | 關閉全螢幕檢視，回到網格列表 |

### 主畫面 (Gallery) 操作
| 操作方式 | 功能說明 |
| :--- | :--- |
| **網格排數滑桿/輸入框** | 即時調整列表圖片顯示的欄數 (支援 1~10排，輸入框無上限) |
| **主介面縮放滑桿** | 針對單排模式快速微調圖片大小 (支援 1~150%，輸入框無上限) |
| **向下/向上捲動** | 自動隱藏/喚出頂部控制列 |

## 🤝 相容性

支援所有基於 Chromium 核心的現代瀏覽器 (Google Chrome, Microsoft Edge, Opera, Brave 等) 以及 Safari、Firefox 最新版本。支援常見影像格式如 `JPG`, `PNG`, `GIF` (支援動態), `WEBP` 等。支援帶有 Alpha 通道的半透明圖片完美顯示。

# 🌌 本地圖片瀏覽器 (Local Image Browser)

這是一個**完全免安裝、零伺服器依賴**的輕量級本地圖片瀏覽器。只需一個單一的 HTML 檔案，點擊兩下即可在瀏覽器中開啟。專為處理擁有數千、數萬張圖片的超大容量資料夾而設計，提供極致流暢的沉浸式看圖體驗。

## ✨ 核心特色功能

* 🚀 **極致效能 (防當機設計)**：導入 `IntersectionObserver` 無限捲動與延遲載入 (Lazy Loading) 技術。即使資料夾內有十萬張圖片，也能在 1 秒內瞬間掃描完成，並只渲染畫面可見的圖片，絕不吃光記憶體。
* 📁 **自動穿透子資料夾**：支援 HTML5 `webkitdirectory`，選取母資料夾（如 `Downloads`）後，會自動抓出所有深層子資料夾內的圖片。
* 🎛️ **無段式自訂網格**：提供 1~10 排的滑桿切換，並支援手動輸入突破 10 排限制，打造專屬的百圖牆。
* 👁️ **沉浸式 UI 設計**：
  * 頂部選單向下捲動時自動隱藏，向上滑動即時喚出。
  * 燈箱全螢幕模式下，所有按鈕與標題皆採用 95% ~ 99% 的高透明度與毛玻璃特效，將視覺干擾降至最低。
  * 左右切換區擴大至螢幕兩側 25% 滿版範圍，實現「盲點」切換。
* 🔍 **看圖神器級操控 (支援滑鼠與鍵盤)**：
  * **滾輪縮放**：在全螢幕模式下，支援滑鼠滾輪自由放大/縮小圖片 (最高 10 倍)。
  * **滑鼠拖曳**：圖片放大後，可直接按住滑鼠左鍵自由拖曳 (Pan) 查看細節，並已修復連點反藍 BUG。
* 📐 **三種滿版模式**：
  * **自動滿版**：預設模式，完整顯示整張圖片。
  * **高度滿版**：適合橫向風景圖或插畫。
  * **寬度滿版 (漫畫模式)**：自動貼齊左右邊緣，適合閱讀 Webtoon 等長條型漫畫，配合滑鼠拖曳極度流暢。

## 🛠️ 使用方式

1. 下載本專案中的 `Local Image Browser.html` 檔案。
2. 對著檔案點擊兩下，使用 Google Chrome、Microsoft Edge 等現代瀏覽器開啟。
3. 點擊畫面左上角的 **「📁 選擇資料夾」**。
4. 在彈出的視窗中，選擇你存放圖片的資料夾（例如 `D:\Downloads`）。
   > *⚠️ 瀏覽器安全性提醒：此時瀏覽器會跳出「要上傳檔案到這個網站嗎？」的警告。請放心點擊「上傳」，本程式完全運行於本地端，**絕對不會**將你的圖片上傳至網路，也**不會**在硬碟中製造暫存檔佔用空間，僅是在記憶體中建立唯讀的讀取通道。*
5. 開始享受極致流暢的看圖體驗！

## ⌨️ 快捷鍵與操作指南

| 操作方式 | 功能說明 |
| :--- | :--- |
| **滑鼠滾輪** (全螢幕) | 放大 / 縮小當前圖片 |
| **按住滑鼠左鍵** (全螢幕) | 拖曳平移已放大的圖片或長條漫畫 |
| **點擊畫面左側 25%** | 切換至上一張圖片 |
| **點擊畫面右側 25%** | 切換至下一張圖片 |
| **鍵盤 `→` (右方向鍵)** | 切換至下一張圖片 |
| **鍵盤 `←` (左方向鍵)** | 切換至上一張圖片 |
| **鍵盤 `ESC`** | 關閉全螢幕檢視，回到網格列表 |
